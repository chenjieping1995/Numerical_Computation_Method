README
===========================
本文件内容为段雅丽老师数值计算方法课程的实验代码，均采用C/C++实现。
****

具体包含了：
------
1.lagrange多项式插值与newton多项式插值 选取 等距分割和chebyshev分割 的对比(多项式插值.cpp)  
2.实现列主元消元法计算方程组(gauss-row.cpp)  
3.实现gauss-seidel迭代功能(gauss-seidel.cpp)  
4.实现jacobi迭代功能(jacobi.cpp)  
5.用四阶Runge-Kutta法求解常微分方程初值问题(四阶Runge-Kutta法.cpp)
